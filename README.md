# Lista de Tareas en JavaScript

Pequeña aplicación para agregar, completar y eliminar tareas usando HTML, CSS y JS puro.

## Funcionalidades
- Agregar tareas
- Marcar como completada
- Eliminar tareas

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript ES6
